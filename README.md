## Installation

```bash

```

Or:

```bash
curl -sL https://joinme.sh | bash
```

## Usage

```bash
# Login with SSH key
msh login

# Post something
msh post "Hello, Mesh!"

# View your feed
msh feed

# Follow someone
msh follow @alice
```
